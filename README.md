# Lost&Found Docs

Техническая документация MVP Lost&Found, оформленная в формате docs as code на Docusaurus.

## Локальный запуск

```bash
cd my-website
npm install
npm run start
```

## Сборка

```bash
cd my-website
npm run build
```

## Публикация

Документация публикуется через GitHub Actions в ветку `gh-pages`.

- Репозиторий: https://github.com/LizaErdakova/lostfound-docs
- GitHub Pages: https://LizaErdakova.github.io/lostfound-docs/
