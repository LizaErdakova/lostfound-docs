## Назначение

REST API описывает взаимодействие интерфейса Lost&Found MVP с backend для трех ключевых экранов: лента объявлений, карточка объявления и создание объявления.

## Redocusaurus

Полная интерактивная спецификация доступна на отдельной странице: [OpenAPI Lost&Found](/api/lostfound).

## Исходный файл спецификации

Спецификация вынесена в файл `api_specs/lostfound-openapi.yaml`, чтобы Redocusaurus мог собрать из него отдельную API-документацию.

## Основные группы endpoints

- Справочники: получение категорий.
- Объявления: лента, карточка, создание объявления.
- Медиа: загрузка фотографий.
